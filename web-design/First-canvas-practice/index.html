I want to set the color which does not exist in the swatches, how can I change the color to the one I want? Don't worry, just look at the solutions provided below. It's pretty simple and easy!

1. Use color name. For example, input "red" to draw a line with the color of an apple.
2. Use color hex number. For example, input "#7CFC00" for LawnGreen.
3. Use the form "rgb(R, G, B)" to set the color by RGB color model. For example, input "rgb(255, 255, 255)" to draw a snow white line.
4. Use "rgba(R, G, B, Opacity)" to set a more powerful color with opacity. For example, input "rgba(0, 153, 204, 0.5)" to draw glasses.
